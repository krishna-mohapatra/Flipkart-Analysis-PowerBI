# Flipkart-Analysis-PowerBI
Developed an end-to-end E-commerce analytics dashboard using the Flipkart dataset, covering data cleaning, transformation, and modeling using PowerQuery. The project analyzes key metrics, including sales, pricing, discounts, and seller performance, to generate actionable business insights and support data-driven decision-making.
<img width="1468" height="732" alt="FKT-Homepage" src="https://github.com/user-attachments/assets/4f0cadbe-8f9a-4b63-85b2-f90b2c5ca67c" /> 

# 🎯 Problem Statement
The objective of this project is to analyze Flipkart e-commerce data to understand key business metrics, including revenue, discounts, product performance, and customer trends, enabling data-driven decision-making.

# 🛠️ Tools & Technologies
<ul>
  <li>Power BI</li>
  <li>SQL</li>
  <li>Excel</li>
</ul>

## 2. Product Performance Dashboard
<p align="center">

<p><b>💡Key Insights: </b></p>
